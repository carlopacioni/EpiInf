Epiinf

BEAST 2 package for performing inference on epidemiological parameters using the exact likelihood.

Download this project as a .zip file Download this project as a tar.gz file

What is EpiInf?

EpiInf is a package for performing inference of stochastic compartmental epidemiological model parameters using particle filtering to compute the prior probability density of a tree. Joint inference of prevalence trajectories is also performed.

EpiInf is currently at the early pre-release stage, so some things may not work perfectly and there is still very little in the way of documentation. Even the name may change!

Installation

To install EpiInf, follow these steps: 1. Install the most recent version of BEAST 2 2. Open the BEAUti programme. 3. Select "Manage Packages" from the "File" menu. 4. Click the "Package repositories" button at the bottom of the package manager dialog box. 5. Choose "Add URL" 6. Enter http://tgvaughan.github.io/EpiInf/package.xml, click OK, then select Done to return to the package manger. 7. EpiInf should now be in the list of packages. Select this and then click Install/Upgrade. 8. Once the successful installation is reported, close the package manager and RESTART BEAUti.

Usage

From BEAUti, load a sequence alignment and set up your analysis as usual. From the Priors panel, select the EpiTreePrior. This allows you to select from the available compartmental models and choose parameters to infer or fix. At this point, choosing File->Save will produce an XML file that you can run using BEAST which will perform the analysis.
